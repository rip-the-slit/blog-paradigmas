<!DOCTYPE>

Indica al navegador el tipo de documento HTML para evitar el "Quirk mode" (modo no estándar).

Código:

<!DOCTYPE html>

<html>

Es el componente raíz. Define el inicio y fin del documento web.

Código:

<html lang="es">
          <!-- Todo el contenido va aquí -->
        </html>

<head>

Cabecera del documento. Contiene metadatos, título y enlaces a recursos externos (CSS/JS).

Código:

<head>
          <title>Mi Página</title>
          <meta charset="UTF-8">
        </head>

<body>

Define el cuerpo del documento. Contiene absolutamente todo lo que el usuario verá en pantalla.

Código:

<body>
          <h1>Bienvenido</h1>
          <p>Este es el contenido visible.</p>
        </body>

Resultado Visual:

Bienvenido

Este es el contenido visible.

<header>

Contenedor para contenido introductorio o menús de navegación superior.

Código:

<header style="background: #eee; padding: 10px;">
          <h2>Logo de la Empresa</h2>
          <nav>Inicio | Contacto</nav>
        </header>

Resultado Visual:

Logo de la Empresa

Inicio | Contacto

<section>

Define secciones genéricas y temáticas en un documento.

Código:

<section style="border: 1px solid #ccc; padding: 10px;">
          <h3>Últimas Noticias</h3>
          <p>Contenido de la noticia...</p>
        </section>

Resultado Visual:

Últimas Noticias

Contenido de la noticia...

<div>

Contenedor genérico de bloque. Ideal para agrupar elementos y aplicarles CSS.

Código:

<div style="background: lightblue; padding: 10px; border-radius: 5px;">
          <h4>Tarjeta de Perfil</h4>
          <p>Desarrollador Web</p>
        </div>

Resultado Visual:

Tarjeta de Perfil

Desarrollador Web

<span>

Contenedor en línea. Aplica estilos a fragmentos sin romper el párrafo.

Código:

<p>El precio es <span style="color: red; font-weight: bold;">$100</span> dólares.</p>

Resultado Visual:

El precio es $100 dólares.

<script>

Incrusta o enlaza código JavaScript.

Código:

<script>
          console.log("Hola mundo");
        </script>

<h1> a <h6>

Encabezados jerárquicos.

Código:

<h1>Título h1</h1>
        <h3>Título h3</h3>
        <h6>Título h6</h6>

Resultado Visual:

Título h1

Título h3

Título h6

<p>

Define un párrafo.

Código:

<p>Este es un párrafo de ejemplo.</p>

Resultado Visual:

Este es un párrafo de ejemplo.

<br>

Inserta un salto de línea forzado.

Código:

<p>Línea uno.<br>Línea dos.</p>

Resultado Visual:

Línea uno.
Línea dos.

<hr>

Línea horizontal separadora.

Abajo
