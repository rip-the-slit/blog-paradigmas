¿Qué es el DOM?

El DOM (Document Object Model) es una interfaz de programación para los documentos web. Representa la página para que los programas puedan cambiar la estructura, el estilo y el contenido del documento. El DOM representa el documento como nodos y objetos; de esta forma, los lenguajes de programación como JavaScript pueden conectarse a la página.

En el DOM, cada etiqueta HTML es un Nodo Un punto de intersección o conexión dentro del árbol del DOM. Un nodo puede ser un elemento HTML, un atributo o incluso texto plano. que forma parte de una estructura de árbol gigantesca.

Selección de Nodos

Para manipular un elemento HTML, primero necesitamos seleccionarlo desde JavaScript. Estos son los métodos principales que nos proporciona el objeto document:

  • getElementById(): Selecciona un único elemento por su atributo id.
  • querySelector(): Devuelve el primer elemento que coincida con el selector CSS especificado.
  • querySelectorAll(): Devuelve todos los elementos que coincidan con un selector CSS en forma de una lista de nodos (NodeList). 
// Seleccionando elementos del DOM
const titulo Principal = document.getElementById('titulo');
const primerParrafo = document.querySelector('.parrafo');
const todosLosBotones = document.querySelectorAll('button');

Modificación de Elementos

Una vez que tienes un nodo seleccionado, puedes modificar su contenido, sus atributos o sus estilos CSS directamente desde JavaScript.

Contenido de Texto y HTML

Propiedades como textContent e innerHTML permiten leer o sobrescribir el interior de una etiqueta. textContent solo inserta texto seguro, mientras que innerHTML permite procesar nuevas etiquetas HTML.

Modificando Clases y Estilos

La propiedad classList es la forma más eficiente de añadir, eliminar o alternar clases CSS dinámicamente. 

// Modificar contenido
tituloPrincipal.textContent = "Nuevo Título Dinámico";

// Añadir HTML interno
primerParrafo.innerHTML = "Texto con <strong>negritas</strong>";

// Trabajar con clases CSS
primerParrafo.classList.add('texto-destacado');
primerParrafo.classList.remove('oculto');
primerParrafo.classList.toggle('activo');

Creación y Eliminación de Nodos

JavaScript no solo puede modificar lo que ya existe, sino que también puede construir partes completamente nuevas de la página o destruir elementos.

  • createElement(tag): Crea un nuevo nodo de elemento HTML en memoria.
  • appendChild(nodo): Añade un nodo como el último hijo de un elemento padre.
  • remove(): Elimina el elemento actual del DOM. 
// 1. Crear el elemento en memoria
const nuevoLi = document.createElement('li');
nuevoLi.textContent = "Elemento creado con JS";

// 2. Seleccionar al padre y agregarlo
const lista = document.querySelector('ul');
lista.appendChild(nuevoLi);

// Eliminar un elemento
const elementoViejo = document.querySelector('.viejo');
elementoViejo.remove();

Eventos y Event Listeners

Las páginas web modernas son interactivas gracias a los eventos. Un evento es cualquier acción que ocurre en el sistema, como un clic del ratón, presionar una tecla, o el simple hecho de que la página termine de cargar.

El método más seguro y poderoso para reaccionar a estos eventos es addEventListener(). Este método le indica al navegador que "escuche" una acción específica en un elemento y ejecute una función (conocida como callback) cuando esa acción suceda. 

const botonGuardar = document.getElementById('btn-guardar');

botonGuardar.addEventListener('click', () => {
  alert('¡Formulario guardado con éxito!');
  botonGuardar.classList.add('exito');
});

El Objeto Evento

Cuando ocurre un evento, JavaScript pasa automáticamente un Objeto Evento (e) Contiene toda la información sobre lo que acaba de ocurrir: qué tecla se presionó, las coordenadas del clic, o qué elemento detonó la acción. a la función callback. Este objeto es crucial para acciones avanzadas, como prevenir comportamientos por defecto. 

const formulario = document.querySelector('form');

formulario.addEventListener('submit', (e) => {
  // Evita que la página se recargue al enviar el formulario
  e.preventDefault();
  
  console.log('Procesando datos con JS...');
  console.log('Elemento que detonó el evento:', e.target);
});
