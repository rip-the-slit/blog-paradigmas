Conceptos básicos de CSS

CSS (Cascading Style Sheets) es el lenguaje utilizado para definir la presentación visual de un documento HTML. Mientras que HTML se encarga de la estructura y el contenido, CSS controla el diseño, los colores, tamaños, espaciados y la disposición de los elementos en la página. Gracias a CSS es posible separar el contenido de la presentación, lo que facilita el mantenimiento y mejora la organización del código. Esta separación permite modificar completamente la apariencia de un sitio web sin alterar su estructura HTML.

Estructura de un documento CSS

Un documento CSS está compuesto por reglas que siguen una estructura específica formada por un selector y un bloque de declaraciones. Cada declaración incluye una propiedad y un valor, separados por dos puntos y finalizados con punto y coma. El selector indica a qué elemento HTML se aplicarán los estilos definidos. El bloque de declaraciones se encierra entre llaves y puede contener múltiples propiedades. Esta estructura permite aplicar estilos de manera clara, organizada y reutilizable. 

        selector {
            propiedad: valor;
            propiedad: valor;
        }

Formas de aplicar CSS

  • CSS externo: Se define en un archivo separado con extensión .css y se enlaza al documento HTML mediante la etiqueta link dentro del elemento head. Este método permite reutilizar los estilos en múltiples páginas, facilita el mantenimiento y mejora la organización del código. Es la forma más utilizada en proyectos web reales porque mantiene una clara separación entre estructura y diseño.
  • CSS interno: Se escribe dentro de una etiqueta style ubicada en la sección head del documento HTML. Este método es útil cuando se desea aplicar estilos a una sola página sin crear un archivo adicional. Aunque es práctico para proyectos pequeños o pruebas, no es recomendable para sitios grandes debido a la dificultad de reutilización.
  • CSS en línea: Se aplica directamente dentro del atributo style de una etiqueta HTML específica. Este método tiene mayor prioridad en la cascada, pero no es recomendable para el desarrollo estructurado porque mezcla contenido y presentación. Se utiliza principalmente para cambios rápidos o ajustes muy específicos en un elemento individual.

Selectores en CSS

Los selectores en CSS permiten identificar qué elementos HTML recibirán un conjunto específico de estilos dentro de una hoja de estilo. Cada selector tiene una función particular y puede combinarse con otros para crear reglas más específicas y precisas. Existen selectores básicos, combinadores, selectores de atributo, pseudoclases y pseudoelementos, todos con aplicaciones distintas. Comprender cómo funciona cada uno es esencial para escribir código CSS limpio, eficiente y bien estructurado. A continuación se presenta una lista organizada de los principales selectores con su respectiva descripción.

Selectores básicos

  • * (selector universal): Selecciona todos los elementos del documento HTML.
  • elemento (selector de tipo): Selecciona todos los elementos de una etiqueta específica, como p, div o h1.
  • .clase (selector de clase): Selecciona todos los elementos que tienen asignada una clase específica.
  • #id (selector de ID): Selecciona el elemento que tiene un identificador único determinado.
  • selector1, selector2 (selector agrupado): Permite aplicar los mismos estilos a varios selectores al mismo tiempo.

Selectores combinadores

  • A B (descendente): Selecciona todos los elementos B que estén dentro de un elemento A.
  • A > B (hijo directo): Selecciona solo los elementos B que sean hijos directos de A.
  • A + B (hermano adyacente): Selecciona el elemento B que esté inmediatamente después de A.
  • A ~ B (hermano general): Selecciona todos los elementos B que sean hermanos de A y estén después de él.

Selectores de atributo

  • [atributo]: Selecciona los elementos que contienen un atributo específico.
  • [atributo="valor"]: Selecciona los elementos cuyo atributo coincide exactamente con un valor.
  • [atributo~="valor"]: Selecciona elementos cuyo atributo contiene una palabra específica separada por espacios.
  • [atributo|="valor"]: Selecciona elementos cuyo atributo comienza con un valor específico seguido opcionalmente de un guion.
  • [atributo^="valor"]: Selecciona elementos cuyo atributo comienza con el valor indicado.
  • [atributo$="valor"]: Selecciona elementos cuyo atributo termina con el valor indicado.
  • [atributo*="valor"]: Selecciona elementos cuyo atributo contiene el valor indicado en cualquier posición.

Pseudoclases

  • :hover: Aplica estilos cuando el usuario coloca el cursor sobre el elemento.
  • :active: Se activa cuando el elemento está siendo presionado.
  • :focus: Se aplica cuando un elemento recibe el foco del teclado.
  • :visited: Selecciona enlaces que ya han sido visitados.
  • :link: Selecciona enlaces que aún no han sido visitados.
  • :first-child: Selecciona el primer hijo de su elemento padre.
  • :last-child: Selecciona el último hijo de su elemento padre.
  • :nth-child(n): Selecciona el hijo en una posición específica.
  • :nth-of-type(n): Selecciona elementos de un tipo específico según su posición.
  • :not(selector): Selecciona elementos que no coinciden con el selector indicado.
  • :checked: Selecciona elementos de formulario marcados, como checkboxes.
  • :disabled: Selecciona elementos de formulario deshabilitados.
  • :enabled: Selecciona elementos de formulario habilitados.
  • :required: Selecciona campos de formulario obligatorios.
  • :valid: Selecciona elementos con contenido válido según sus restricciones.
  • :invalid: Selecciona elementos con contenido inválido.
  • :root: Selecciona el elemento raíz del documento, generalmente html.

Pseudoelementos

  • ::before: Inserta contenido antes del contenido real del elemento.
  • ::after: Inserta contenido después del contenido real del elemento.
  • ::first-letter: Selecciona la primera letra del texto de un elemento.
  • ::first-line: Selecciona la primera línea del texto.
  • ::selection: Aplica estilos al texto seleccionado por el usuario.
  • ::placeholder: Aplica estilos al texto de marcador de posición en campos de formulario.
  • ::marker: Selecciona los marcadores de listas, como números o viñetas.

Propiedades básicas de formato

Las propiedades en CSS permiten controlar la apariencia visual y el comportamiento de los elementos HTML dentro de una página web. Cada propiedad define un aspecto específico del diseño, como colores, tamaños, márgenes o posiciones. Se escriben dentro de una regla CSS con el formato propiedad: valor y pueden combinarse para lograr diseños complejos. Comprender qué hace cada propiedad es esencial para desarrollar sitios web organizados y visualmente consistentes. A continuación se presentan las propiedades básicas con una breve descripción de su función.

Propiedades de color y fondo

  • color: Define el color del texto del elemento.
  • background: Propiedad abreviada para definir todas las propiedades de fondo en una sola declaración.
  • background-color: Establece el color de fondo de un elemento.
  • background-image: Define una imagen como fondo del elemento.
  • background-repeat: Controla si la imagen de fondo se repite y cómo lo hace.
  • background-position: Especifica la posición inicial de la imagen de fondo.
  • background-size: Determina el tamaño de la imagen de fondo.
  • background-attachment: Indica si el fondo se desplaza con el contenido o permanece fijo.
  • opacity: Define el nivel de transparencia del elemento.

Propiedades de texto y fuente

  • font: Propiedad abreviada para definir varias características de la fuente.
  • font-family: Especifica el tipo de letra del texto.
  • font-size: Define el tamaño del texto.
  • font-style: Establece el estilo de la fuente, como normal o italic.
  • font-weight: Define el grosor del texto.
  • font-variant: Controla variantes tipográficas como small-caps.
  • line-height: Establece la altura de línea o espacio vertical entre líneas de texto.
  • text-align: Alinea el texto horizontalmente.
  • text-decoration: Agrega o elimina decoraciones como subrayado o tachado.
  • text-transform: Controla la transformación del texto en mayúsculas o minúsculas.
  • text-indent: Define la sangría de la primera línea del texto.
  • letter-spacing: Ajusta el espacio entre caracteres.
  • word-spacing: Ajusta el espacio entre palabras.
  • white-space: Controla cómo se manejan los espacios y saltos de línea.

Propiedades del modelo de caja

  • width: Define el ancho del elemento.
  • height: Define la altura del elemento.
  • min-width: Establece el ancho mínimo permitido.
  • max-width: Establece el ancho máximo permitido.
  • min-height: Define la altura mínima permitida.
  • max-height: Define la altura máxima permitida.
  • margin: Propiedad abreviada para definir los márgenes exteriores.
  • margin-top: Define el margen superior.
  • margin-right: Define el margen derecho.
  • margin-bottom: Define el margen inferior.
  • margin-left: Define el margen izquierdo.
  • padding: Propiedad abreviada para definir el espacio interno.
  • padding-top: Define el espacio interno superior.
  • padding-right: Define el espacio interno derecho.
  • padding-bottom: Define el espacio interno inferior.
  • padding-left: Define el espacio interno izquierdo.
  • border: Propiedad abreviada para definir borde, ancho, estilo y color.
  • border-width: Define el grosor del borde.
  • border-style: Define el estilo del borde, como sólido o punteado.
  • border-color: Define el color del borde.
  • border-radius: Redondea las esquinas del borde.
  • box-sizing: Controla cómo se calculan el ancho y alto del elemento.

Propiedades de visualización y posicionamiento

  • display: Define el tipo de caja del elemento, como block o inline.
  • visibility: Controla si el elemento es visible o no.
  • position: Establece el método de posicionamiento del elemento.
  • top: Define la distancia desde la parte superior en elementos posicionados.
  • right: Define la distancia desde la derecha.
  • bottom: Define la distancia desde la parte inferior.
  • left: Define la distancia desde la izquierda.
  • z-index: Controla el orden de apilamiento de elementos.
  • float: Permite que un elemento flote a la izquierda o derecha.
  • clear: Controla el comportamiento frente a elementos flotantes.
  • overflow: Define qué ocurre cuando el contenido excede el tamaño del elemento.
  • overflow-x: Controla el desbordamiento horizontal.
  • overflow-y: Controla el desbordamiento vertical.

Concepto de cascada y prioridad

El término cascading hace referencia al sistema de prioridad que utiliza CSS para aplicar estilos cuando existen múltiples reglas que afectan al mismo elemento. Los estilos en línea tienen mayor prioridad que los definidos en archivos externos o internos. Los selectores de ID tienen mayor especificidad que los de clase, y estos a su vez mayor que los de etiqueta. Además, cuando dos reglas tienen la misma especificidad, se aplicará la que esté definida más abajo en el documento. Comprender este mecanismo es esencial para evitar conflictos en el diseño.
